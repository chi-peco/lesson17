1. は［主題しゅだい］ 2. には／では／とは／へは 3. にも／でも／とも／へも 4. 何なにも／だれも〜ません 5. が［描写びょうしゃ］ まとめ Summary
1. は［文ぶんの主題しゅだい］ は as object-topic marker — changing を → は
EXAMPLES — Let's look first. → まず例文れいぶんを見みてみましょう。
しゅくだいは あした 出だして ください。
Please hand in your homework tomorrow. (homework = topic)
この 本ほんは もう 読よみました。
This book, I have already read. (book = topic)

Normally, the object of a sentence uses the particle 「を」.

→ ふつう、目的語もくてきご（object）には「を」をつかいます。(目的語もくてきご = the thing that receives the action)

But when you want to highlight the object as the topic of the sentence, or contrast it with something else, change 「を」→「は」.

→ 目的語もくてきごを話題わだいにしたり、他ほかのものと対比たいひするとき、「を」を「は」に変かえます。(The basic meaning does not change — only the focus changes.)

💡 The particle 「を」disappears — it is replaced by 「は」.

→ 「を」は消きえます。「は」に入いれ替かわります。(「を」vanishes and 「は」takes its place.)
N[目的語もくてきご] + を → は + V
N = noun（名詞めいし）　V = verb（動詞どうし）
change: 「を」→「は」
映画えいがを 見みました。 → 映画えいがは 見みました。 I watched the movie. → The movie, I watched. (highlighting "movie" as topic)
○ 映画えいがは 見みました。でも、ドラマは 見みませんでした。 The movie I watched. But the drama I did not. — 「は」marks each object as a contrasted topic. → 「は」で対比たいひしている。（Contrasting two objects with は.）
× 映画えいがを 見みました。でも、ドラマを 見みませんでした。 Using 「を」in both sentences loses the contrast. — 「を」では対比たいひが出でない。（No contrast with を.）
EXAMPLE 例文れいぶん
しゅくだいは あした 出だして ください。
Please hand in your homework tomorrow. (homework = highlighted topic)
牛乳ぎゅうにゅうは 飲のみます。でも、ジュースは 飲のみません。
I drink milk. But juice, I don't drink. (contrasting two objects)

Change 「を」to 「は」to highlight the object as a topic.
→「を」を「は」に変かえて、目的語もくてきごを話題わだいにしてみましょう。
＿＿＿＿＿＿＿は ＿＿＿＿＿＿＿ました。
2. には／では／とは／へは particle + は — topic / contrast marker added to other particles
EXAMPLES — Let's look first. → まず例文れいぶんを見みてみましょう。
私わたしの国くにには すし屋やが ありません。
In my country, there are no sushi restaurants. (に + は = には)
この 店みせでは カードが つかえません。
In this shop, you cannot use a card. (で + は = では)

In Lesson 1, you learned that 「は」marks the topic of a sentence. In Section 1 of this lesson, 「を」was replaced by 「は」.

→ L1で「は」が主題しゅだいを示しめすことを学まなびました。セクション１では「を」が「は」に変かわりました。(Lesson 1: topic marker. Section 1: を replaced by は.)

For other particles (に、で、と、へ), the particle stays and 「は」is added after it.

→ に・で・と・へなど、他ほかの助詞じょしには「は」を後うしろにつけます。助詞じょしは消きえません。(The particle stays. 「は」is added on top of it.)
助詞じょし Particle ＋ は 意味いみ Meaning 例れい Example
に には at / in (location) — highlighted 国くにには 海うみが ありません。In my country there is no sea.
で では at / in (action location) — highlighted 学校がっこうでは スマホを つかいません。At school, I don't use my phone.
と とは with (person) — highlighted Aさんとは 話はなしましたが、Bさんとは まだです。I spoke with A, but not with B yet. (contrast)
へ へは to (direction) — highlighted アメリカへは 行いきましたが、カナダへは 行いきませんでした。I went to America, but not to Canada. (contrast)
N ＋ particle ＋ は
に → には ／ で → では ／ と → とは ／ へ → へは
N = noun（名詞めいし）

助詞じょしの基本きほんの使つかい方かた What each particle means on its own

Before adding 「は」, make sure you understand what each particle does by itself. → 「は」をつける前まえに、それぞれの助詞じょしの基本きほんの意味いみを確認かくにんしましょう。

助詞じょし 基本きほんの意味いみ Basic meaning 主おもな用法ようほう Main uses 例れい Example (without は)

shows where something is, when it happens, or who/where it goes to		 ①時とき time (L5)
②対象たいしょう target (L7)
③存在そんざいの場所ばしょ location (L8)
④目的もくてき purpose (L9)

⑥置おく場所ばしょ place to put (L14) 		6時じに おきます。I wake up at 6. 東京とうきょうに つきました。I arrived in Tokyo. 机つくえの上うえに 本ほんが あります。There is a book on the desk.

shows where an action happens, or how it is done (means / method)		 ①動作どうさの場所ばしょ action location (L4)
②方法ほうほう・手段しゅだん means (L5) 		としょかんで 本ほんを 読よみます。I read at the library. (action location) バスで いきます。I go by bus. (means)

📝 に vs で vs へ — 紛まぎらわしい助詞じょし Easily confused particles
  • に (location) → where something exists: 机つくえの上うえに あります。 (It is ON the desk.)
    → ものが「ある・いる」場所ばしょ。(Where something exists.)
  • で (location) → where an action happens: としょかんで 読よみます。 (I read AT the library.)
    → 動作どうさが行おこなわれる場所ばしょ。(Where an action takes place.)
  • へ (direction) → where you are heading: 学校がっこうへ いきます。 (I go TOWARD school.)
    → 移動いどうの方向ほうこう。(Direction of movement.)
○ 日本にほんには 温泉おんせんが たくさん あります。 「に」stays, 「は」is added → 「には」. Japan is highlighted as the topic. → 「に」は残のこり「は」がつく。日本にほんが話題わだいになる。(In Japan, there are many hot springs.)
× 日本にほんは 温泉おんせんが たくさん あります。 The particle 「に」is missing — this sentence loses the location meaning. → 「に」がないと場所ばしょの意味いみが消きえる。(The location meaning disappears without に.)
EXAMPLE 例文れいぶん
私わたしの国くにには すし屋やが ありません。
In my country, there are no sushi restaurants. (に + は)

→ には・では・とは・へはを使つかって、場所ばしょや人ひとを話題わだいにしてみましょう。
＿＿＿＿＿＿＿には ＿＿＿＿＿＿＿が ＿＿＿＿＿＿＿。
3. にも／でも／とも／へも particle + も — "also / too" added to other particles
EXAMPLES — Let's look first. → まず例文れいぶんを見みてみましょう。
木曜日もくようびにも テストが あります。
There is a test on Thursday too. (に + も = にも)
コンビニでも チケットが 買かえます。
You can also buy tickets at a convenience store. (で + も = でも)

In Lesson 3, you learned that 「も」means "also / too" and replaces 「は」or 「が」.

→ L3で「も」が「は」や「が」の代かわりに使つかわれることを学まなびました。("also/too" replaces は or が in simple sentences.)

For other particles (に、で、と、へ), the particle stays and 「も」is added after it.

→ に・で・と・へなどには、「も」を後うしろにつけます。助詞じょしは消きえません。(The particle stays. 「も」is added on top.)

💡 This means "the same thing is true in this case too".

→「この場合ばあいにも同おなじことが言いえる」という意味いみ。("The same applies here too.")
助詞じょし Particle ＋ も 意味いみ Meaning 例れい Example
に にも at / in — also 水曜日すいようびにも クラスが あります。There is also a class on Wednesday.
で でも at / by — also スーパーでも 買かえます。You can buy it at the supermarket too.
と とも with — also Bさんとも 話はなしました。I also talked with B.
へ へも to — also カナダへも 行いきたいです。I also want to go to Canada.
N ＋ particle ＋ も
に → にも ／ で → でも ／ と → とも ／ へ → へも
N = noun（名詞めいし）
📝 Section 2 vs Section 3 — は vs も
  • particle ＋ は → highlights / contrasts that item (the focus is this one vs others)
    → 対比たいひ・強調きょうちょう。「これ」と「他のものほかのもの」を比くらべる。
  • particle ＋ も → adds another item ("also / too" — the same thing applies here too)
    → 追加ついか。「これも同おなじ」という意味いみ。
○ A：「火曜日かようびに テストが ありますか。」 B：「ええ。木曜日もくようびにも ありますよ。」 「にも」adds Thursday as another day when a test occurs. → 「にも」で「木曜日もくようびもテストがある」という追加情報ついかじょうほうを伝つたえている。(A: "Is there a test on Tuesday?" B: "Yes. There is one on Thursday too.")
× B：「ええ。木曜日もくようびには ありますよ。」 「には」would contrast or highlight Thursday — it does not add information about "also". → 「には」だと対比たいひになり、「追加ついか」の意味いみにならない。(には creates a contrast, not an addition.)
EXAMPLE 例文れいぶん
A：「火曜日かようびに テストが ありますか。」
B：「ええ。木曜日もくようびにも ありますよ。」
A: "Is there a test on Tuesday?" B: "Yes. There is one on Thursday too."
✏️ TRY IT! 自分じぶんで書かいてみよう
Add にも / でも / とも / へも to a sentence to say "also / too".
→ にも・でも・とも・へもを使つかって「〜も」の文ぶんを作つくってみましょう。
＿＿＿＿＿＿＿にも ＿＿＿＿＿＿＿が あります。
4. 何なにも／だれも／だれにも／どこも／どこへも〜ません Total negation — "nothing / nobody / nowhere"
EXAMPLES — Let's look first. → まず例文れいぶんを見みてみましょう。
教室きょうしつに だれも いません。
There is nobody in the classroom.
きのう どこへも 行いきませんでした。
Yesterday, I did not go anywhere.

Use these words to say "nothing", "nobody", "nowhere" — a total denial.

→「何なにも」「だれも」「どこも」などは全面的ぜんめんてきな否定ひていを表あらわします。(Total negation — nothing / nobody / nowhere.)

Always use with a negative verb form (〜ません / 〜ませんでした). You cannot use these words with a positive verb.

→ 必かならず否定形ひていけい（〜ません・〜ませんでした）と一緒いっしょに使つかいます。(Must always use with negative verb form.)

How to form: start with the question word, then look at the particle.

→ 疑問詞ぎもんし（question word）の後あとの助詞じょしを確認かくにんします。(Check the particle after the question word.)
• If the particle is 「が」or「を」 → change to「も」 →「が」「を」の場合ばあいは「も」に変かえる。
• For other particles (に、へ, etc.) → add「も」after the particle → その他ほかの助詞じょしには「も」をつける。

何なにも — nothing 何なにも（のこと）

何なに ＋ が／を → 何なにも ＋ ません
○ 冷蔵庫れいぞうこに 何なにも ありません。 「が」is replaced by「も」, negative verb → total negation. → 「が」→「も」、否定形ひていけいで全面否定ぜんめんひてい。(There is nothing in the fridge.)
× 冷蔵庫れいぞうこに 何なにも あります。 「何も」must be used with a negative verb. 「あります」is positive. → 「何なにも」は否定形ひていけいと使つかう。（Cannot use 何も with a positive verb.）
EXAMPLE 例文れいぶん — 何なにも (nothing)
今日きょうは 何なにも 食たべませんでした。
Today, I did not eat anything.

だれも — nobody だれも（のこと）

だれ ＋ が／を → だれも ＋ ません
○ 教室きょうしつに だれも いません。 「が」→「も」, negative verb → "nobody is here". → 「が」→「も」＋否定形ひていけいで「だれもいない」という意味いみ。(There is nobody in the classroom.)
× だれも 来きました。 「来ました」is positive. 「だれも」needs a negative verb → 「だれも 来ませんでした」. → 「来きました」は肯定形こうていけいなのでNG。（だれも must be used with ません.）
EXAMPLE 例文れいぶん — だれも (nobody)
パーティーに だれも 来きませんでした。
Nobody came to the party.

だれにも — to nobody だれにも（のこと）

だれ ＋ に ＋ も → だれにも ＋ ません   (「に」stays, add「も」)
○ この ことは だれにも 言いいませんでした。 「に」stays, 「も」is added → "to nobody". → 「に」は残のこり「も」をつける。（I did not tell anyone about this.）
× この ことは だれも 言いいませんでした。 「に」is dropped — the meaning changes. 「だれにも」= "to nobody". 「だれも」= "nobody (as subject)". → 「に」がないと意味いみが変かわる。（Without に, the meaning changes.）
EXAMPLE 例文れいぶん — だれにも (to nobody)
この ひみつは だれにも 話はなさないでください。
Please don't tell this secret to anyone.

どこも — nowhere どこも（のこと）

どこ ＋ が／を → どこも ＋ ません   (「が／を」→「も」)
EXAMPLE 例文れいぶん — どこも (nowhere)
休日きゅうじつは どこも こんでいます。
On holidays, everywhere is crowded. (どこも + positive = "everywhere / all places")
病気びょうきで どこも 行いけませんでした。
Because I was sick, I could not go anywhere. (どこも + negative = "nowhere")
⚠️ どこも — special note 注意ちゅうい

「どこも」+ positive verb can mean "everywhere". The total-negation meaning ("nowhere") requires a negative verb.

→「どこも」は肯定形こうていけいと使つかうと「どこでも（everywhere）」の意味いみになることがある。全面否定ぜんめんひてい（nowhere）には否定形ひていけいが必要ひつよう。(With positive verb = "everywhere". With negative = "nowhere".)

どこへも — to nowhere どこへも（のこと）

どこ ＋ へ ＋ も → どこへも ＋ ません   (「へ」stays, add「も」)
○ きのう どこへも 行いきませんでした。 「へ」stays, 「も」added, negative verb → "did not go anywhere". → 「へ」は残のこり「も」をつける。否定形ひていけいで「どこへも行いかない」。(Yesterday, I did not go anywhere.)
× きのう どこも 行いきませんでした。 「どこも」is used for "exists somewhere" / "nowhere (in place)". For movement, use 「どこへも」. → 移動いどう（go / come）には「どこへも」を使つかう。（For movement verbs, use どこへも.）
EXAMPLE 例文れいぶん — どこへも (to nowhere)
きのう どこへも 行いきませんでした。
Yesterday, I did not go anywhere.
何なにも ／ だれも ／ どこも ／ だれにも ／ どこへも ＋ Vません（でした）
V = verb（動詞どうし）

＿＿＿＿＿＿＿も ＿＿＿＿＿＿＿ませんでした。
5. が［描写びょうしゃ］ が for description — noticing something for the first time

「が」has several uses. Today you will learn use ④. → 「が」にはいくつかの用法ようほうがあります。今日きょうは④を学まなびます。

# 用法ようほう Use 習ならった回かい Lesson 例れい Example
① 感情かんじょう・能力のうりょくの対象たいしょう
object of feelings / abilities — "like / good at ~"		 L7 音楽おんがくが 好すきです。I like music.
② 主語しゅご subject marker (existence)
"there is ~ / ~ exists"		 L8 部屋へやに 猫ねこが います。There is a cat in the room.
③ 逆接ぎゃくせつ "but" / contrast
connects two contrasting ideas		 L11 日本語にほんごは好すきですが、むずかしいです。I like Japanese, but it is difficult.
④ 描写びょうしゃ description — noticing something new
✦ TODAY — pointing out / describing what you notice		 L17 ✦ 雨あめが ふっています。It is raining. (you just noticed it)
EXAMPLES — Let's look first. → まず例文れいぶんを見みてみましょう。
雨あめが ふっています。
It is raining. (you looked outside and noticed it)
空そらが きれいですね。
The sky is beautiful, isn't it! (you notice the sky right now)

When you notice something and want to describe or point it out, use 「が」for the subject — not 「は」.

→ 気きづいたことを描写びょうしゃするとき、主語しゅごに「が」を使つかいます。「は」ではありません。(When pointing out something you notice, use が for the subject.)

This often happens when describing a scene, or reporting something you just noticed for the first time.

→ 景色けしきの描写びょうしゃや、今いま気きづいたことを伝つたえるときに使つかいます。(Describing a scene or something you just noticed.)
📝 は vs が — key difference 重要じゅうようなちがい
  • 〜は → the topic is already known to the listener. You are talking about it.
    → すでに知しっている話題わだいについて話はなす。(雨あめは つめたいです = talking about rain as a known topic)
  • 〜が → you are pointing out something new you just noticed.
    → 今いま気きづいたことを指ゆびさす。(雨あめが ふっています = noticing it is raining right now)
○ 雨あめが ふっています。 You looked outside and noticed it is raining → 「が」describes the scene. → 気きづいた場面ばめんを描写びょうしゃするので「が」を使つかう。(It is raining. [you just noticed])
× 雨あめは ふっています。 「は」suggests "rain" is an established topic — this sounds unnatural when you are simply noticing / reporting the weather. → 「は」は既知きちの話題わだいに使つかう。気きづきの場面ばめんでは不自然ふしぜん。(は is for known topics — unnatural for noticing.)
EXAMPLE 例文れいぶん
雨あめが ふっています。
It is raining. (You noticed it is raining and are describing the scene.)
外そとに 子こどもが います。
There is a child outside. (You noticed a child — pointing it out.)

📝 Key points — 17回目かいめ
  • を → は: The particle 「を」disappears and 「は」replaces it. Meaning stays the same — focus changes.
    →「を」が「は」に変かわる。意味いみは同おなじで、焦点しょうてんが変かわる。
  • particle ＋ は (には・では等): The particle stays, 「は」is added. = highlight / contrast.
    → 助詞じょしは残のこり「は」をつける。対比たいひ・強調きょうちょう。
  • particle ＋ も (にも・でも等): The particle stays, 「も」is added. = "also / too".
    → 助詞じょしは残のこり「も」をつける。追加ついか（also/too）。
  • 何なにも・だれも・どこも（へも）: Always use with a negative verb (ません / ませんでした).
    → 必かならず否定形ひていけい（ません・ませんでした）と一緒いっしょに使つかう。
  • が［描写びょうしゃ］: When you notice something new → use 「が」. When talking about a known topic → use 「は」.
    → 気きづきの描写びょうしゃは「が」。既知きちの話題わだいは「は」。
